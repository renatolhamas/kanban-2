# Workflow: Resposta a Guidance (TBD-PRE)

**INSTRUÇÕES PARA O ANTIGRAVITY:**

1. **Objetivo:** Reestruturar perguntas de orientação (guidance) feitas pelo agente DEV (geralmente após um Pre-Flight) para incluir contexto, opções e recomendações imediatas.
2. **Execução:**
   - Identifique os itens/perguntas que o agente DEV acabou de solicitar orientação (ex: itens listados após "provide guidance on the following items").
   - Reapresente cada um desses itens na íntegra, mas formatados da seguinte forma:
     - **Situação:** Uma breve explicação do contexto técnico ou funcional.
     - **Opções/Alternativas:** Lista de caminhos ou soluções possíveis.
     - **Recomendação:** A opção mais adequada para o momento atual da **Kanban Messaging Platform** e seguindo os padrões do **AIOX**.
3. **Persona:** Atue como um consultor técnico sênior/arquiteto, garantindo que o DEV tenha clareza para seguir com a implementação.

**Uso:** Execute `/tbd-pre` quando o DEV solicitar "guidance" sobre uma story ou tarefa.
