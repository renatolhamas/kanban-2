9. Regras Executivas (Do's and Don'ts)

✅ Fazer (Do)

  • • Use margens espaçosas para sensação "Premium"
  • • Use Verde Esmeralda com parcimônia
  • • Use pesos negritos para dados numéricos
  • • Em dúvida: aumente o espaçamento

❌ Não Fazer (Don't)

  • • NÃO use preto puro. Use cinza escuro.
  • • NÃO use bordas sólidas de 1px
  • • NÃO use cinzas "sujos"
  • • NÃO comprima a interface
