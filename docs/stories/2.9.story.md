**Copy-to-Clipboard Implementation:**

📌 **Primary API: `navigator.clipboard.writeText()`** ✅ 100% Compatible in this stack
- Install: No installation required (native Web API)
- **Browser Support:** Chrome 63+, Firefox 53+, Safari 13.1+, Edge 79+ (all modern)
- **CRITICAL Requirement:** Requires transient user activation (user must click within ~2 minutes)
- **Example:**
  ```typescript
  button.addEventListener('click', async () => {
    try {
      await navigator.clipboard.writeText('var(--color-primary)');
      showToast('✅ Copiado!');
    } catch (err) {
      showToast('❌ Erro ao copiar');
    }
  });
  ```

📌 **Fallback for IE11:** `document.execCommand('copy')`
- No installation needed (native DOM API)
- **Browser Support:** IE11, older Edge versions (rarely encountered in modern apps)
- **Important:** Only use as fallback, slower than Clipboard API
- **Example Fallback Implementation:**
  ```typescript
  const copyToClipboard = async (text: string): Promise<boolean> => {
    // Try modern API first
    if (navigator.clipboard?.writeText) {
      try {
        await navigator.clipboard.writeText(text);
        return true;
      } catch (err) {
        console.warn('Clipboard API failed, trying fallback', err);
      }
    }

    // Fallback for older browsers
    const textarea = document.createElement('textarea');
    textarea.value = text;
    textarea.style.position = 'fixed'; // Prevent page scroll
    textarea.style.opacity = '0'; // Hide from view
    document.body.appendChild(textarea);
    textarea.select();
    
    try {
      const success = document.execCommand('copy');
      document.body.removeChild(textarea);
      return success;
    } catch (err) {
      document.body.removeChild(textarea);
      console.error('Copy failed:', err);
      return false;
    }
  };

  // Usage with error handling:
  button.addEventListener('click', async () => {
    const success = await copyToClipboard('var(--color-primary)');
    showToast(success ? '✅ Copiado!' : '❌ Falha ao copiar');
  });
  ```
