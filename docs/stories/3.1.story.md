## Dev Notes

### API Route Pattern (Source: architecture/5-backend-architecture-api-routes-middleware.md)

All `/api/*` routes are protected by middleware stack:

```typescript
// Pattern for /api/settings/evo-go/qr/route.ts
import { auth } from "@/lib/middleware/auth";
import { tenantIsolation } from "@/lib/middleware/tenant-isolation";

export async function POST(req: Request) {
  try {
    // 1. Verify JWT, extract user_id from token
    const user = await auth(req);
    
    // 2. Ensure tenant_id is set and matches request
    const { tenantId } = await tenantIsolation(req, user);
    
    // 3. Call Evo GO API with retry logic
    const qrData = await callEvoGoCreateInstance(tenantId);
    
    // 4. Update tenants table
    await supabase
      .from("tenants")
      .update({
        evolution_instance_id: qrData.instance_id,
        connection_status: "pending"
      })
      .eq("id", tenantId);
    
    // 5. Return success response
    return Response.json({
      qr_code: qrData.qr_code,
      instance_id: qrData.instance_id,
      expires_at: qrData.expires_at
    }, { status: 200 });
  } catch (error) {
    // Error handling logic
    return handleEvoGoError(error);
  }
}
```

**Reference:** [5-backend-architecture-api-routes-middleware.md](../../docs/architecture/5-backend-architecture-api-routes-middleware.md)

### Evo GO Integration Details (Source: architecture/evo-go-api-mapping.md)

**Endpoint**: `POST /instance/create` (Evo GO API)

**Request**:
```json
{
  "apiKey": "EVO_GO_API_KEY"
}
```

**Response** (Success 200):
```json
{
  "instance_id": "abc123def456",
  "qr_code": "data:image/png;base64,...",
  "phone": "+55...",
  "status": "connecting",
  "expires_at": "2026-04-12T12:30:00Z"
}
```

**Status Mapping**:
- `open` (Evo GO) -> `connected` (Sistema)
- `connecting` (Evo GO) -> `connecting` (Sistema)
- `close` (Evo GO) -> `disconnected` (Sistema)
- Any failure -> `error`

**Error Responses**:
- 401: Invalid API key
- 429: Rate limit exceeded (Implement hybrid retry 3x then notify user)
- 500: Evo GO server error
- Timeout: Network timeout after 5 seconds

**HMAC Validation** (for webhook responses):
- Algorithm: SHA256
- Header: `X-Signature`
- Secret: **Instance Token** (Option A - uniqueness per tenant)
- Use: `crypto.timingSafeEqual()` for comparison

**Reference:** [evo-go-api-mapping.md](../../docs/architecture/evo-go-api-mapping.md)
