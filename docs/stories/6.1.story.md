# Story 6.1: Chat Modal UI (View History)

## Executor Assignment

executor: "@dev"
quality_gate: "@architect"
quality_gate_tools: ["vitest", "lint", "typecheck"]

---

## Story Statement

**As an** attendant,
**I want to** open a chat modal when clicking on a conversation card,
**so that** I can view the message history in a familiar WhatsApp-style interface.

---

## Scope

**IN:**
- `ChatModal` component using shadcn/ui Dialog.
- `MessageList` component for displaying message bubbles (left/right).
- `MessageInput` component with text area and static send/media buttons.
- Header with contact information (name, phone).
- Static `KanbanTransferDropdown` and `ArchiveButton` placeholders.
- Basic "bubble" styling using design system colors.
- Integration with `ConversationCard` to trigger the modal.

**OUT:**
- Real-time message updates (Story 6.5).
- Sending logic (Story 6.2).
- Message status tracking (Sent/Delivered/Read) — Phase 3.
- Media upload functionality (Phase 3).
