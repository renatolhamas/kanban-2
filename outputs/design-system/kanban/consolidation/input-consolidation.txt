ACCESSIBILITY CONSIDERATIONS
─────────────────────────────

✅ Touch target size:
   - px-4 py-2.5 = ~48px height (good, exceeds 44px guideline)

✅ Focus indicator:
   - focus:ring-2 + focus:border-blue-500 provides clear visual feedback
   - Ring color (blue-500/20) is subtle but visible

✅ Color contrast:
   - Placeholder (gray-400) on gray-50 = ~4.5:1 (WCAG AA ✓)
   - Text (black) on gray-50 = ~12:1 (WCAG AAA ✓)
   - Focus ring (blue-500) on gray-50 = ~3.5:1 (barely AA, acceptable for focus state)

✅ Disabled state:
   - Disabled text (gray-500) on gray-50 = ~3.2:1 (WCAG AA for disabled ✓)
